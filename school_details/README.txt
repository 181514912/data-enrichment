This folder contains the code to scrap the details to CBSE schools.

All the details have been scrapped from "cbseaff.nic.in/cbse_aff/schdir_Report/userview.aspx"

File_1.py and File_2.py scripts are used to fetch the urls of all the CBSE schools from the above mentioned link.

File_3.py script is used store the details of the schools in the csv file format.
